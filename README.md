# GUMIMUG

## ゲームの内容
グミでできた主人公が走って跳ねる！爽快ランニングアクション！
おちているお菓子を集めて、またはオブジェクトを壊して、世界一のお菓子を目指せ！

![画面イメージ](docs/images/game_image01.png)

## 操作説明
例
Wキーで前進
A、Dキーで旋回
スペースキーを長押しで踏ん張り
スペースキーを離すと踏ん張った分だけジャンプ
空中でスペースキーを押すと高速落下
エンターキーで攻撃
